\doxysubsection{Detailed Description}
Defines the problem interface and a concrete Advection-\/\+Diffusion-\/\+Reaction (ADR) problem. 

This file contains the abstract base class {\ttfamily Problem\+Interface} which allows the solver to be decoupled from specific physics. It also provides a concrete implementation {\ttfamily ADRProblem} using the Method of Manufactured Solutions (MMS) for verification.
