\doxysubsection{Detailed Description}
Copy data for the {\ttfamily Work\+Stream} assembly. 

This struct holds the results of the local integration (cell matrix, cell rhs, and DoF indices). This data is passed from the worker threads to the main thread (copier) to be added to the global system.
