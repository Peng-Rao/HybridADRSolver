\doxysubsection{Detailed Description}
\subsubsection*{template$<$int dim, int fe\+\_\+degree$>$\newline
class Hybrid\+ADRSolver\+::\+Matrix\+Free\+Solver$<$ dim, fe\+\_\+degree $>$}
Matrix-\/free solver with hybrid MPI+threading parallelization. 

This class implements a matrix-\/free finite element solver that\+:
\begin{DoxyItemize}
\item Uses Matrix\+Free for efficient operator application
\item Supports hybrid MPI (distributed) + threading (shared memory) parallelization
\item Implements GMRES/\+CG with Jacobi preconditioning
\item Achieves better memory efficiency and cache utilization than matrix-\/based
\end{DoxyItemize}
