\doxysubsection{Detailed Description}
\subsubsection*{template$<$int dim, int fe\+\_\+degree, typename Number = double$>$\newline
class Hybrid\+ADRSolver\+::\+ADROperator$<$ dim, fe\+\_\+degree, Number $>$}
Matrix-\/free operator for the ADR problem. 

This class implements the action of the system matrix on a vector without explicitly storing the matrix. The computation uses\+:
\begin{DoxyItemize}
\item Sum factorization for efficient tensor-\/product evaluation
\item Vectorization over multiple cells (SIMD)
\item Hybrid MPI+threading parallelization
\end{DoxyItemize}

The weak form implemented is\+:  $ a(u,v) = (\mu \nabla u, \nabla v) + (\beta \cdot \nabla u, v) + (\gamma
u, v) $
